(1.6.1.3.2.1) Richness and diversity estimates Before count filtering

Counts are rarefied to the lowest library size (2161), abundance-based and incidence-based alpha diversity indices and richness estimates are computed (if requested). This is repeated multiple times (n=400), and the results are averaged. Beta diversity matrix is also computed by averaging over multiple rarefications. Incidence-based estimates are computed on sample pools split by metadata attribute visit, and in each repetition, samples are also stratified to balance the number of samples at each level of the grouping variable. Dimensions of input count matrix (67 x 147).
